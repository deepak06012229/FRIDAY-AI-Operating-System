# Roadmap – FRIDAY AI Operating System

## Completed (v1.0 – Stable)

- Memory System (SQLite persistence with automatic categorisation)
- Ollama Integration (local LLM inference)
- Workflow Engine (declarative JSON workflows)
- Voice System (Edge‑TTS with queue management)
- Dashboard (HUD circle, telemetry, tabbed UI)

## Planned (v1.1)

- Better Personality (enhanced Chief‑addressed responses and tone)
- Improved Memory Grounding (refined fact injection and retrieval heuristics)
- Enhanced Dashboard States (additional HUD colours and micro‑animations)

## Future (v2.0)

- ElevenLabs Support (high‑quality voice synthesis)
- Browser Automation (headless browsing and form filling)
- Scheduling System (cron‑style workflow scheduling)
- Advanced Memory Search (semantic lookup across stored facts)
- Multi‑Profile Support (multiple user profiles with separate memories)

*All entries are verified against the current codebase; planned and future items are forward‑looking and do not claim implementation.*
