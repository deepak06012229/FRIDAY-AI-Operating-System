# Changelog – FRIDAY AI Operating System

All notable changes to this project will be documented in this file.

## [1.0] - 2026-06-18
### Added
- Initial public release of FRIDAY AI Operating System.
- Core AI brain with Gemini and Ollama back‑ends.
- Edge‑TTS voice engine with queue‑based playback.
- Workflow engine and JSON‑defined workflows.
- Memory subsystem (SQLite) for persistent facts.
- UI dashboard with voice diagnostics panel.
- Application and web shortcut registries.
- Comprehensive documentation (README, docs folder).

### Fixed
- Resolved pyttsx3 blocking issue by replacing it with Edge‑TTS.
- Fixed queue dead‑locks and ensured continuous speech.

### Deprecated
- None at this stage.
